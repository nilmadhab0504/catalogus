# CATALOGUS Image processing 

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/nilmadhab0504/catalogus
   cd catalogus
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the server:
   ```bash
   npm run dev
   ```
4. Run the worker script:
    ```
    node worker.mjs
    ```
